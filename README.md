## 자바 웹을 다루는 기술 실습 프로젝트

### pro18 : MVC2 기반 게시판 통합 프로젝트 (JSP / Servlet)

이 프로젝트는 **「자바 웹을 다루는 기술」 17강(Model2 방식)**에서 학습한 개념과 기능들을 하나의 웹 애플리케이션으로 통합한 **MVC2 기반 게시판 프로젝트**입니다.

회원 관리와 게시판 기능을 중심으로 실제 웹 서비스의 흐름에 맞게 구현하였으며, 로그인 및 권한 처리 등 부수적인 기능까지 함께 포함하고 있습니다.

---

## 프로젝트 목적

* JSP / Servlet 기반 MVC2(Model2) 구조 이해 및 적용
* 요청 흐름(Request → Controller → Service → DAO → View)에 대한 실습
* 회원 기능과 게시판 기능을 하나의 프로젝트로 통합
* 단순 CRUD를 넘어 실제 서비스에 가까운 구조 경험

---

## 주요 구현 기능

### 1. 회원 관리 (Member)

* 회원 가입
* 로그인 / 로그아웃
* 회원 정보 조회
* 회원 정보 수정
* 회원 삭제
* HttpSession 기반 로그인 세션 관리

### 2. 게시판 (Board)

* 게시글 목록 조회
* 게시글 상세 조회
* 게시글 작성
* 게시글 수정
* 게시글 삭제
* 답글 작성 (계층형 게시글 구조)

### 3. 공통 / 부가 기능

* MVC2(Model2) 아키텍처 적용
* Servlet을 이용한 Front Controller 패턴
* JSP(View) + JSTL / EL 사용
* 로그인 여부에 따른 기능 접근 제어
* Redirect / Forward 흐름 구분 처리

> ※ 페이징 기능은 책의 다음 단계로 추후 추가 예정

---

## 프로젝트 구조 (개요)

* Controller : 요청 URL에 따라 기능 분기 처리
* Service    : 비즈니스 로직 처리
* DAO        : 데이터베이스 접근
* DTO(VO)    : 데이터 전달 객체
* View(JSP)  : 화면 출력 담당

---

## 개발 환경

* Language : Java
* Web      : JSP / Servlet
* Server   : Apache Tomcat
* JDK      : JDK 17
* DB       : Oracle (실습 환경)
* IDE      : Eclipse

## 정리

본 프로젝트는 **책에서 학습한 개념을 실제 동작하는 하나의 웹 애플리케이션으로 완성**하는 것을 목표로 하였으며,

MVC2 구조, 세션 기반 로그인 처리, 게시판 CRUD 및 답글 기능까지 포함한 **기초 웹 프로젝트의 종합 결과물**입니다.
